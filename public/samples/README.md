# Sample labels

The "Try a sample label" link on the homepage fetches `public/samples/compliant-bourbon.jpg`. This image is not committed (see top-level README for rationale).

To enable the link, place a JPEG of a fully-compliant alcohol label at this path. Public-domain product photography from Wikimedia Commons works well, or generate one with an image model.

When the file is missing, the homepage shows a clear inline error rather than failing silently.
